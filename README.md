# Energy Trading Data Analysis (SQL Project)

## Overview
This project demonstrates SQL-based data analysis on an energy trading dataset. It focuses on trade performance, profit/loss analysis, and volume trends across different commodities.

## Objective
To analyze trading activity and extract business insights using SQL queries such as aggregation, filtering, and ranking.

## Dataset
Simulated energy trading dataset containing:
- Trade ID
- Trade Date
- Commodity Type (Gas, Power, Oil)
- Volume
- Price
- Profit & Loss (PnL)

## Key Business Questions
- Which commodity generates the highest profit?
- What is the total trading volume by commodity?
- Which trades are most profitable?
- How does PnL vary across trades?

## SQL Techniques Used
- SELECT queries
- GROUP BY aggregations
- ORDER BY sorting
- Basic analytics logic

## Sample Insights
- Gas trading shows higher profitability in sample dataset
- Oil trades have higher volume but variable returns
- Power trades show mixed PnL patterns

## Tools Used
- SQL
- GitHub for version control
